# Sales Data Analysis with Pandas & Matplotlib

## Project Overview

This project analyzes a real-world online sales dataset using **Python**. The goal is to extract meaningful insights such as identifying top-selling products and high-value customers through data cleaning, aggregation, and visualization techniques.

The analysis helps understand sales patterns and customer purchasing behavior.

---

## Project Highlights

* Cleaned and analyzed **49,000+ sales records**.
* Handled missing values and corrected data types for accurate analysis.
* Created a new metric: **Total Sales = Quantity × Unit Price × (1 − Discount)**.
* Identified:

  * Top 10 products by total sales
  * Top 10 customers by purchase volume
* Visualized insights using **Matplotlib** with bar and line charts.

## How to Run the Project

### 1. Clone the Repository

```
git clone https://github.com/umarbasha-ai/SALES-DATA-ANALYSIS
cd SALES-DATA-ANALYSIS
```

### 2. Install Required Libraries

```
pip install -r requirements.txt
```

### 3. Run the Analysis Script

```
python src/sales-data-analysis.py
```
